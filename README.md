# Simple-To_Do-App

A lightweight proof-of-concept to-do app built with plain HTML, CSS, and JavaScript.

## Features

- Add tasks with low, medium, or high priority
- Block empty submissions with inline validation
- Mark tasks as complete without hiding them
- Delete tasks individually
- Persist tasks locally with `localStorage`
- Show an active task counter and empty state

## Run locally

Open [index.html](./index.html) in a browser on your machine.
